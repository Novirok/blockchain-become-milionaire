# blockchain-become-milionaire
Blockchain API, vůbec tomu nerozumím a nevím jestli to vůbec funguje.


### musíte si nainstalovat flask
    pip install flask

## Spusťe skript python. Server poběží na http://localhost:5000.

## Interakce přes API:

- Získání řetězce: GET http://localhost:5000/chain

- Vytěžení nového bloku: GET http://localhost:5000/mine

 - Odeslání transakce: POST http://localhost:5000/transactions/new
